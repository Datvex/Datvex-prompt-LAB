# Datvex prompt LAB

欢迎来到 **Datvex prompt LAB**。这是一个专业且结构化的仓库，包含了针对多种 AI 系统和专业任务设计的高质量英语提示词（Prompts）。

本集合包含从全球 AI 社区（包括 GitHub 仓库、Discord 频道和其他专业资源）精心策划的提示词，以及专为此实验室开发的原创提示词。

---

## 📂 导航
请根据任务和用例浏览库。目前，所有链接均指向官方策划的数据库：

*   [**按用途浏览**](./NAVIGATION_PURPOSE.md) – 查找按领域分类的提示词（编程、心理学、营销等）。
*   [**✨ AI 提示词搜索（新品）**](https://datvex.github.io/Datvex-prompt-LAB/) – 智能 AI 驱动的搜索，快速找到适合您需求的完美提示词。

---

## 🚀 路线图
我们正在不断发展，以提供最好的提示词资源。

- [x] **阶段 1：** 推出官方策划的数据库（当前）。
- [ ] **阶段 2：** 扩展类别和高级提示词工程模板。
- [ ] **阶段 3：** **社区数据库。** 将为用户增加专用空间，用于分享、测试和存储他们自己的提示词。
