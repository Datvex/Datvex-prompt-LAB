# Datvex prompt LAB

**Datvex prompt LAB**에 오신 것을 환영합니다. 이 저장소는 다양한 AI 시스템의 전문적인 작업을 위해 설계된 고품질 영어 프롬프트를 담고 있는 전문적이고 구조화된 저장소입니다.

이 컬렉션에는 GitHub 저장소, Discord 채널 및 기타 전문 리소스를 포함한 글로벌 AI 커뮤니티의 다양한 출처에서 엄선된 프롬프트와 본 LAB을 위해 특별히 개발된 독창적인 프롬프트가 포함되어 있습니다.

---

## 📂 탐색
작업 및 사용 사례별로 라이브러리를 탐색해 보세요. 현재 모든 링크는 공식 큐레이션 데이터베이스로 연결됩니다.

*   [**목적별 탐색**](./NAVIGATION_PURPOSE.md) – 분야별(프로그래밍, 심리학, 마케팅 등)로 분류된 프롬프트를 찾아보세요.
*   [**✨ AI 프롬프트 검색 (신규)**](https://datvex.github.io/Datvex-prompt-LAB/) – AI 기반 스마트 검색으로 필요에 맞는 완벽한 프롬프트를 빠르게 찾아보세요.

---

## 🚀 로드맵 (Roadmap)
우리는 최고의 프롬프트 리소스를 제공하기 위해 끊임없이 진화하고 있습니다.

- [x] **1단계:** 공식 큐레이션 데이터베이스 출시 (현재).
- [ ] **2단계:** 카테고리 확장 및 고급 프롬프트 엔지니어링 템플릿 도입.
- [ ] **3단계:** **커뮤니티 데이터베이스.** 사용자가 자신의 프롬프트를 공유하고 테스트하며 저장할 수 있는 전용 공간이 추가될 예정입니다.
