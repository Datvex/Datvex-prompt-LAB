# Datvex prompt LAB

**Datvex prompt LAB** へようこそ。当リポジトリは、複数の AI システムにおける専門的なタスク向けに設計された、高品質な英語プロンプトを体系的にまとめた専門的なリポジトリです。

このコレクションには、GitHub リポジトリ、Discord チャンネル、その他の専門的リソースなど、グローバルな AI コミュニティのさまざまなソースから慎重に厳選されたプロンプトのほか、この LAB 専用に開発された独自のプロンプトが含まれています。

---

## 📂 ナビゲーション
タスクやユースケースに合わせてライブラリを閲覧してください。現段階では、すべてのリンクは公式のキュレート・データベースに繋がっています。

*   [**目的別に探す**](./NAVIGATION_PURPOSE.md) – 分野別（プログラミング、心理学、マーケティングなど）に分類されたプロンプトをご覧いただけます。

---

## 🚀 ロードマップ
私たちは、最高のプロンプト・リソースを提供するために絶えず進化しています。

- [x] **フェーズ 1:** 公式キュレート・データベースのリリース（現在）。
- [ ] **フェーズ 2:** カテゴリの拡張と高度なプロンプトエンジニアリング・テンプレートの追加。
- [ ] **フェーズ 3:** **コミュニティ・データベース。** ユーザーが独自のプロンプトを共有、テスト、保存できる専用スペースが追加される予定です。
